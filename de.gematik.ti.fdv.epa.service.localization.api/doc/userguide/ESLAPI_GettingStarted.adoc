=== Getting Started
==== Build setup
To use ePA Service Localization API library in a project, you need just to include following dependency:

.Gradle dependency settings to use ePA Service Localization API library
[source, groovy, subs="attributes+"]
dependencies {
    implementation group: 'de.gematik.ti.fdv', name: 'epa.service.localization.api', version: '{version_ESLAPI}'
}

.Maven dependency settings to use ePA Service Localization API library
[source, xml, subs="attributes+"]
<dependencies>
    <dependency>
        <groupId>de.gematik.ti.fdv</groupId>
        <artifactId>epa.service.localization.api</artifactId>
        <version>{version_ESLAPI}</version>
    </dependency>
</dependencies>
